# 💎 SKCurrency — v1.0
**Підтримувана версія гри:** `1.16.5+` (Залежності: `PlaceholderAPI`)

---

### 📖 Про плагін
**SKCurrency** — це легке та надійне рішення для створення додаткової ігрової валюти на вашому сервері. Плагін ідеально підходить для реалізації систем "Донат-валюти", "Токенів" або "Кристалів", які працюють незалежно від основної економіки (Vault).

Всі дані зберігаються в базі даних **SQLite**, що гарантує високу швидкість роботи та цілісність балансів гравців навіть при великому онлайні.

---

### ✨ Ключові особливості

* **🗄 Надійна база даних**
  Використання SQLite забезпечує миттєвий доступ до балансів та автоматичне створення таблиць при першому запуску.
* **📈 Система ТОП-10**
  Вбудована логіка для відстеження найбагатших гравців сервера. Виводьте лідерів у меню або на голограми.
* **🖇 Повна інтеграція з PAPI**
  Плагін постачається з вбудованим розширенням для PlaceholderAPI, що дозволяє відображати баланс у будь-яких плагінах (Scoreboard, Tab, Chat).
* **🛠 Потужне керування для адмінів**
  Повний набір команд для маніпуляції балансом: видача, зняття, встановлення та повне скидання бази.
* **🎨 Гнучкі повідомлення**
  Підтримка колірних кодів (`&`) та повна кастомізація префікса та всіх системних повідомлень через `config.yml`.

---

### 💻 Командний центр

| Команда | Опис | Доступ |
| :--- | :--- | :--- |
| `/skd look <player>` | Переглянути баланс іншого гравця | `skcurrency.admin` |
| `/skd give <player> <amount>` | Видати валюту гравцеві | `skcurrency.admin` |
| `/skd take <player> <amount>` | Забрати валюту у гравця | `skcurrency.admin` |
| `/skd reset <player>` | Обнулити баланс конкретного гравця | `skcurrency.admin` |
| `/skd resetall` | Повне очищення всієї бази даних | `skcurrency.admin` |
| `/skd reload` | Перезавантажити конфігурацію плагіна | `skcurrency.admin` |

---

### 🧩 PlaceholderAPI (Приклади)

* `%skcurrency_balance%` — Відобразити баланс гравця.
* `%skcurrency_top_name_1%` — Ім'я гравця на 1-му місці в топі.
* `%skcurrency_top_balance_1%` — Баланс гравця на 1-му місці в топі.
